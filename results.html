Each of these sequences of 4 (beginning with the tail) is then converted to a number using NHEX with base 16: (db41)16= (56129)10
Then the number 13459 is divided modulo by 39 (or 18 for the last iteration), and the remainder is taken as one of the balls: 56129 mod 39 = 1439 x 39 + 8
Consequently, the first ball: 8
We repeat this procedure 5 times and discard the number that is already in the set. For the last ball, we simply use mod 18 instead of mod 39.
